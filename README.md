# Объединение .xls файлов

Приложение Streamlit объединяет несколько файлов Excel 97-2003 (.xls) в один итоговый файл `Акция ОБЩИЙ.xls` с унифицированными колонками и заполнением категории по ключевым словам.

## Запуск локально

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Запустите приложение:

```bash
streamlit run app.py
```

## Деплой в Streamlit Cloud

1. Создайте новый репозиторий и загрузите файлы проекта.
2. В Streamlit Cloud выберите репозиторий и файл `app.py` как точку входа.
3. Убедитесь, что `requirements.txt` находится в корне проекта.

## Примечание по колонке "Категория"

Категория определяется по файлу `keywords`, который должен лежать рядом с `app.py`. В первой колонке — ключевое слово, во второй — название категории. Если ключевое слово встречается в названии товара, категория берется из второй колонки. При отсутствии файла или совпадений колонка будет пустой.

## Примечание по колонке "Фото"

В Excel 97-2003 изображения обычно являются встроенными объектами и не хранятся в значениях ячеек. Приложение переносит значения ячеек и гиперссылки, поэтому встроенные картинки без ссылок могут не перенестись.
