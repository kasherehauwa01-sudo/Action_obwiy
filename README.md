# Объединение .xlsx файлов

Приложение Streamlit объединяет несколько файлов Excel (.xlsx) в один итоговый файл `Акция ОБЩИЙ.xlsx` с унифицированными колонками и заполнением категории по ключевым словам.

## Запуск локально

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Запустите приложение:

```bash
streamlit run app.py
```

## Деплой в Streamlit Cloud

1. Создайте новый репозиторий и загрузите файлы проекта.
2. В Streamlit Cloud выберите репозиторий и файл `app.py` как точку входа.
3. Убедитесь, что `requirements.txt` находится в корне проекта.

## Примечание по колонке "Категория"

Категория определяется по файлу `keywords` (возможные форматы: без расширения, `.xlsx`, `.csv`, `.txt`), который должен лежать рядом с `app.py`. В первой колонке — ключевое слово, во второй — название категории. Если ключевое слово встречается в названии товара, категория берется из второй колонки. При отсутствии файла или совпадений колонка будет пустой.

## Примечание по колонке "Фото"

В `.xlsx` изображения могут быть встроенными объектами листа. Приложение извлекает такие изображения и вставляет их в итоговый файл, а если изображений нет, пробует скачать их по URL из колонки \"Фото\".
